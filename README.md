# NutriScan-Web-Scraping-

## Overview

NutriScan is a FastAPI-based application that combines **web scraping**, **data extraction**, and **ingredient scoring** to help users understand the quality of food products.

Given a product name, the application:

1. Searches for the product online.
2. Scrapes ingredient information .
3. Extracts and cleans ingredient data.
4. Evaluates ingredient quality using a scoring algorithm.
5. Returns a health score along with ingredient details.

The goal is to provide a simple way to assess packaged food products based on their ingredient composition.

---

## Features

* Search products by name
* Web scraping for product information
* Ingredient extraction and cleaning
* Ingredient quality scoring
* FastAPI REST API endpoints

* Structured JSON responses

## API Endpoint

### Analyze Product

```http
POST /analyze
```

Request:

```json
{
    "product_name": "Maggi Masala Noodles"
}
```

Example Response:

```json
{
    "product_name": "Maggi Masala Noodles",
    "ingredients": [
        "Wheat Flour",
        "Palm Oil",
        "Salt",
        "Sugar",
        "Spices"
    ],
    "score": 72,
    "category": "Moderate"
}
```

## Ingredient Scoring Logic

The scoring system evaluates ingredients based on predefined rules.

Examples:

### Positive Ingredients

* Whole Grains
* Oats
* Nuts
* Seeds
* Natural Spices

### Negative Ingredients

* Artificial Colors
* Artificial Flavors
* High Fructose Corn Syrup
* Excess Sugar
* Preservatives

A final score is generated by balancing positive and negative ingredient contributions.

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/product-ingredient-analyzer.git

cd product-ingredient-analyzer
```

Create virtual environment:

```bash
python -m venv venv
```

Activate virtual environment:

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```
